变量

与js一样，变量是var定义的。

比如说

func vara(){   
    var a,c int=5,6   
    var b string="asds"  
    fmt.Println(a,b,c)
}

在变量名字后面加类型，当然你不加他也能自动辨别，但是我寻思我个c程序员还是习惯加一个。

当然还有更简单的方法定义

xxxxxxxxxx
func vara(){ 
    a,b:=5,"asd"  
    fmt.Println(a,b)
}

加个冒号定义，简单是简单了，说实话我有点接受不来，而且句尾也不用;非常诡异，第一次定义要加:，冒号这个定义方式一定要在函数里面，函数外会出问题

在函数外面的var定的变量叫做包内变量，不叫全局变量，只要在包内使用好不会危险。而且go有个机制就是定义的变量一定要用，不用就会红线。

内建变量类型

整形里多了一些int+数字的组合，go里没有long啥的，加数字就完了，加ptr是指针，rune是char，因为这个char在其他语言中引起了不少问题，complex是复数，就是根号-1的那个东西。

xxxxxxxxxx
func vara(){ 
    a:=3+4i  
    fmt.Println(cmplx.Abs(a))
}

这个输出当然是5

go的数据类型转化是强制的，就是说它不能像c一样，你是个float，我用的时候给你个警告就给你变成int了，go不行，要转换的时候必须用int()强制转化，不转化是用不了的（怎么就这么严格这么麻烦）

常量

常量没啥变化就是const

xxxxxxxxxx
func vara(){ 
    const filename="asd.txt"  
    const a,b=3,4 
    c:=int(math.Sqrt(a*a+b*b))   
    fmt.Println(c,filename)
}

当然也是可以指定变量类型的，不指定的话就是交给它判断了

函数

这个函数就很直接

xxxxxxxxxx
func div(a int,b int,) (int ,int){
    return a/b,a%b
}

当然返回值也可以命名数值，都没差

xxxxxxxxxx
func div(a int,b int,) (q int ,r int){
    q=a/b
    r=a%b
    return q,r
}

返回的值定义了下面当然就不用定义了

当你接返回的值时，如果你就想要一个，那不要的整个下划线代替就完了

xxxxxxxxxx
func div(a int,b int,) (q int ,r int){
    q=a/b
    r=a%b
    return q,r
}
func main() {
    a,_:=div(13,4)
    fmt.Println(a)
}

一般情况下不要乱用返回俩值的，俩值一般情况下一个是要用的，一个是错误返回值error，这是因为你不能让别人用你的项目，出错的时候给个panic直接给停止了，这个太难看。

比如说

xxxxxxxxxx
func div(a int,b int,) (int ,int,error){
    if b!=0 {
        return a/b,a%b,nil
    }else {
        return 0,0,fmt.Errorf("除数为0")
    }
}
func main() {
    a,_,err:=div(13,0)
    fmt.Println(a,err)
}

在go里面，函数没有重载这些花里胡哨的，只有粗暴的。只有可变参数列表

numbers就是里面有多少个数都无所谓
